WgytBot

WgytBot commands:

Usage doesn't include the prefix. The prefix is w! or an @mention.

ping

This command pings the bot.

Usage: ping
help

This command DM'S the user help with the bot.

Usage: help
pingme

This command pings the user who said the command.

Usage: pingme
source

This tells the user about the source code.

Usage: source
christmas

This command celebrates christmas.

Usage: christmas
gtg/got to go

This command says @message-author has to go.

Usage: gtgUsage2: got to go
status

This command says the status of https://bot.wgyt.tk.

Usage: statusUsage2: got to go
