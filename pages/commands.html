WgytBot

WgytBot commands:

Usage doesn't include the prefix. The prefix is w! or an @mention.

ping

This command pings the bot.

Usage: ping
help

This command DM'S the user help with the bot.

Usage: help
pingme

This command pings the user who said the command.

Usage: pingme
source

This tells the user about the source code.

Usage: source
christmas

This command celebrates christmas.

Usage: christmas
gtg

This command says @message-author has to go.

Usage: gtg
botstatus

This command says the status of https://bot.wgyt.tk.

Usage: botstatus
webstatus

This command says the status of the link provided.

Usage: webstatus https://example.com
repeat

This command repeates the user.

Usage: repeat Hi
funnyquote

This command repeates the user in a funny way.

Usage: funnyquote Hi @author
kick

This command kicks the user mentioned.

Usage: kick @mention
ban

This command bans the user mentioned.

Usage: ban @mention
unban

This command unbans the user mentioned.

Usage: unban @mention
